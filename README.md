# 🏢 Authority Buildings Management System

A full-featured Java Desktop Application developed to manage authority buildings, departments, and maintenance operations using a structured relational database system.

---

## 📌 Project Background

This project was developed as my Secondary School Graduation Project.

Although the project was officially assigned to a 10-member team, the full system analysis, database design, UI implementation, and backend development were completed independently.

The system was built from scratch, covering planning, architecture design, database modeling, and full implementation.

---

## 🎯 Project Objective

The goal of this system is to digitize and organize building management operations within an authority environment by:

- Managing buildings and departments
- Tracking maintenance activities
- Organizing employee records
- Storing structured data securely in a relational database
- Improving administrative workflow efficiency

---

## 🏗️ System Architecture

The application follows a structured layered architecture:

### 1️⃣ Presentation Layer
- Java Swing GUI
- Interactive forms
- User input validation

### 2️⃣ Business Logic Layer
- Application rules
- Data processing
- Workflow handling

### 3️⃣ Data Access Layer
- JDBC integration
- SQL queries
- Oracle database connectivity

## 🗄️ Database Design

The database was designed using relational modeling principles.

### Core Tables

- Buildings
- Departments
- Employees
- Maintenance Records
- Administrative Units

### Database Features

- Primary & Foreign Keys
- Relational Constraints
- Normalized Schema
- Structured Relationships
- Indexed searchable fields

The database schema was planned before implementation using ERD modeling techniques.

### 🔹 Building Management

## 🚀 Installation & Setup

1. Install Java JDK (8 or higher)
2. Install Oracle Database (XE recommended)
3. Create the required database schema using provided SQL scripts
4. Update JDBC connection credentials in the source code
5. Run the project using NetBeans or any compatible Java IDE
